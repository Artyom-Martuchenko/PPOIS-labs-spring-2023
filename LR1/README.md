## ATM model

### Install

Clone repository:

```bash
git clone https://github.com/AlexeyTerleev/labs-sem4/tree/main/PPOIS/lab1
```



### Run

```bash
cd lab1
python3 main.py
```

### Flags

* -atm
    * -insert-card (card number)
    * -extract-card
    * -input-pin (PIN-code)
    * -balance
    * -put-money (amount)
    * -get-money (amount)
* -bank
    * -register-acc (login) (password)
    * -login (login) (password)
    * -logout
    * -register-card (PIN-code)
