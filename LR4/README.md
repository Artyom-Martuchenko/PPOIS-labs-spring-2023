## PPOIS lab4

### Instalation

Clone repository:
```bash
git clone https://github.com/AlexeyTerleev/labs-sem4/tree/main/PPOIS/lab4
```

Install dependencies:
```bash
pip install -r requirements.txt
```

### Run

#### GUI
```bash
cd lab4
python3 gui_main.py
```

#### CLI
```bash
cd lab4
python3 cli_main.py
```
